AnimationNodeTimeScale¶

Inherits: AnimationNode < Resource < RefCounted < Object

A time-scaling animation node used in AnimationTree.

Description¶

Allows to scale the speed of the animation (or reverse it) in any children AnimationNodes. Setting it to 0.0 will pause the animation.
