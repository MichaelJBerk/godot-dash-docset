GradientTexture¶

Inherits: Texture < Resource < Reference < Object

Gradient-filled texture.

Description¶

GradientTexture uses a Gradient to fill the texture data. The gradient will be filled from left to right using colors obtained from the gradient. This means the texture does not necessarily represent an exact copy of the gradient, but instead an interpolation of samples obtained from the gradient at fixed steps (see width). See also CurveTexture.
